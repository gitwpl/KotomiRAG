## Introduce

RAG练手项目, 项目主题是通过提交动漫字幕, 通过RAG更好地分析人物以及动漫剧情

## Project analysis

第一个部分目前只测试了一个, 当出现多个选项的时候能不能兼容还没有测试, 需要先提交字幕文件才能选择

![动漫选择](./images/Snipaste_2026-04-17_19-18-46.png)

提交的文件目前只支持单文件txt传输

![提交字幕文件](./images/Snipaste_2026-04-17_19-25-26.png)

聊天的功能需要配合“动漫选择”模块, 需要先点击动漫选择的某个选项才能正常使用

![聊天](./images/Snipaste_2026-04-17_19-26-33.png)
